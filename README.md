# note-app
Full Stack Developer

1. **Frontend:**

        ```
            cd frontend
        ```
        
        ```
            npm install
        ```

        ```
            npm run dev
        ```

2. **APIApplication:**
    - Open Visual Basic Studio 
    - Cofig connection string
    
    ```
      "ConnectionStrings": {
        "DefaultConnection": "Server=localhost\\SQLEXPRESS;Database=NotesDb;Trusted_Connection=True;TrustServerCertificate=true"
       },
    ```

    ```
        run the project 
    ```
